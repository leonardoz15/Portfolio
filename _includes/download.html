Current Projects:

  • GatorQuery

    I contributed to the GatorQuery course evaluation tool as part of a software project for CMPSC 481 Software Innovation. This project was written primarily in Python supporting a Flask web framework for an accessible web interface. As a member of a 6 person development team, I was tasked with creating the login authorization system which used a flask form to query an sqlite3 database. If the user was already registered, their information and status would be retrieved, redirecting them to either a student or teacher session based on their status. If there wasn't yet an account associated with the login credentials, the user would be redirected to a registration page where they can input a new account into the database.
